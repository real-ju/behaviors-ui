Select 选择器 ​

基本使用 ​

html
<be-select
  title="标题"
  v-model:visible="visible"
  mode="single"
  v-model:value="selectValue"
  :list="selectList"
>
</be-select>
ts
import { ref } from 'vue';

const visible = ref(false);
const selectValue = ref<number>();
const selectList = ref([
  { label: '选项1', value: 1 },
  { label: '选项2', value: 2 },
  { label: '选项3', value: 3 }
]);

TIP

Select组件基于 DataPopup 和 SelectView 封装，更多示例请参考对应组件。
